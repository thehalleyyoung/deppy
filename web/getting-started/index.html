Overview

Getting started

Installation and first commands for both real-adoption mode and spectacular prompt-driven generation.

On this page

There are two main ways to enter deppy: check code you already have, or ask it to synthesize something ambitious.

Install

pip3.11 install -e .

Editable mode is the intended development workflow. It exposes the deppy command and keeps the hybrid and agent code live from the checkout.

Real-adoption path

deppy .
deppy check src/ --format sarif
deppy check src/ --incremental

This mode is designed to work on code that was not written for deppy.

Spectacular path

deppy --ideation --orchestration --lean --strict --verbose "build a verified trading system that discovers a novel market idea and proves the risk-critical core"

This path searches for a typed new idea, builds a typed orchestration cover, generates local sections, and glues them into a project.

What the flags mean

  • --ideation: run the ideation engine, treating cross-domain ideas as typed analogies.
  • --orchestration: generate through a TT/AG cover with module roles and gluing obligations.
  • --lean: emit Lean obligations and run the discharge path.
  • --strict: require a stricter success condition.
  • --verbose: show the whole pipeline in the terminal.
