Works on raw Python

No annotations or contracts required. Feed deppy ordinary `.py` files and it reasons over control flow, data flow, and semantic obligations.

Finds real bug classes

Division by zero, null dereference, deadlock, SQL injection, use-after-close, index errors, and more all show up as local-to-global mismatches.

Checks equivalence

deppy compares programs by descent instead of examples, giving a mathematical criterion for when two implementations glue to the same behavior.

Explains every result

The output is not just a warning. It identifies the obstructing sites, the violated transition, and the minimum independent fixes implied by the cocycle rank.

Examples

Readable from the command line or a browser

Detect a bug

from deppy.render.bug_detect import detect_bugs

report = detect_bugs("""
def normalize(values):
    total = sum(values)
    return [v / total for v in values]
""")

for obs in report.obstructions:
    print(obs.bug_type, obs.line, obs.description)

Check equivalence

from deppy.equivalence.pipeline import (
    EquivalencePipeline,
    EquivalencePipelineConfig,
)

pipeline = EquivalencePipeline(EquivalencePipelineConfig())
result = pipeline.run(program_f, program_g)
print(result.verdict.value)

The mathematics

This is the algorithm, not branding

Programs are sheaves

Each program site carries local semantic constraints. Compatible local sections glue to a global interpretation.

Bugs live in H¹

If local constraints disagree on overlaps, the obstruction class witnesses the bug and localizes the missing fix.

Equivalence is descent

Two implementations are equivalent when their transported sections agree on all overlaps and the induced cocycle is trivial.

Proofs are explicit artifacts

Structural obligations can be discharged with Z3 and translated to Lean, making the trust story visible instead of implicit.
